# 🤖 JARVIS

**J.A.R.V.I.S — Just A Rather Very Intelligent System**

A Python-based desktop assistant designed to automate common computer tasks through a simple and interactive command-line interface. JARVIS can launch applications, perform web searches, open websites, and execute custom commands while providing a clean user experience inspired by the AI assistant from Iron Man.

---

## ✨ Features

* 🌐 Launch Google Chrome
* 🔍 Search anything on Google
* 📺 Open YouTube instantly
* 📷 Activate the system camera
* 🎵 Execute custom secret commands
* 💻 Cross-platform support (Windows, macOS, Linux)
* ⚡ Fast and lightweight
* 🛡️ Basic error handling for improved reliability
* 🤖 Interactive JARVIS-style responses

---

## 🖥️ Available Commands

| Command        | Description                   |
| -------------- | ----------------------------- |
| chrome         | Launch Google Chrome          |
| camera         | Open system camera            |
| youtube        | Open YouTube                  |
| google <query> | Search Google                 |
| ZeroDayX       | Execute custom secret command |
| help           | Display command menu          |
| exit / quit    | Shutdown JARVIS               |
